# Object Avoidance Robot

A Python-based autonomous robot developed using the **Webots Robotics Simulator**. The project demonstrates autonomous navigation by detecting obstacles using camera input and dynamically adjusting the robot's movement to avoid collisions.

---

## Overview

The objective of this project was to design and implement an autonomous robot capable of navigating its environment without human intervention. The robot continuously processes visual data from its onboard camera, detects coloured obstacles, and makes real-time navigation decisions based on their position.

This project strengthened my understanding of robotics, computer vision, algorithm design, and autonomous decision-making while applying Python in a simulated robotics environment.

---

## Features

- Autonomous obstacle detection
- Camera-based colour recognition
- Real-time navigation decisions
- Dynamic left/right obstacle avoidance
- Continuous environment monitoring
- Implemented using Python in Webots

---

## Technologies Used

- Python
- Webots Robotics Simulator

---

## How It Works

1. The robot captures images using its onboard camera.
2. Each frame is analysed to detect coloured obstacles.
3. The position of the detected obstacle is determined.
4. The controller decides whether to:
   - Continue moving forward
   - Turn left
   - Turn right
5. The process repeats continuously, allowing autonomous navigation.
